## 📞 Quick Reference

### Commands
```bash
# Setup
bash setup.sh              # Mac/Linux
setup.bat                  # Windows

# Start backend
cd backend && npm run dev

# Start frontend
cd frontend && npm run dev

# Build for production
cd frontend && npm run build
```
